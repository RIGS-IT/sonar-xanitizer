Название урока: Обращение к скрытым ресурсам.

Тема для изучения:

Как производить обращение к скрытым ресурсам

Как работают такие атаки:

Эта техника используется хакерами для обращения к тем ресурсам, ссылок на которые на сайте нет, но доступ к которым никак не ограничен. Одним из примеров такой техники является затирание части URL для того чтоб просмотреть содержимое незащищённой директории.

Основные цели и задачи:

* Вашей целью является угадывание URL интерфейса конфигурации.
* Ссылка на него видна только управляющему персоналу.
* Приложение не проверяет наличие соответствующих привилегий при доступе к нему
