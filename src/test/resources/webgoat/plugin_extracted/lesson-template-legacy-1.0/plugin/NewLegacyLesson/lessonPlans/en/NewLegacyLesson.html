Lesson Plan Title: How to create a legacy lesson

Concept / Topic To Teach:

This text block is coming from the resources/plugin/lessonPlans/en/NewLegacyLesson.html file. WebGoat will take the text between the start/stop instructions comments in this file and insert them as instructions in the running lesson. You can also use the getInstructions() method in the NewLegacyLesson.java file to return instructions. This allows users to modify the lesson instructions without modifying the code. We recommend using this file for instructions to allow easy translation to other languages.

General Goal(s):

The user should be able to create and deploy a new lesson.
