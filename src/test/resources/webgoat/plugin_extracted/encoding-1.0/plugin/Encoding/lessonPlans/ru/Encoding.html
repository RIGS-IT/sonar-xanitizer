Название урока: Использование основной кодировки

Тема для изучения:

По множеству причин содержимое веб-приложения может храниться в нескольких разных кодировках.

Основные цели и задачи:

Данный урок предназначается для тех, кто знаком с понятием кодировок и понимает чем они отличаются друг от друга.
